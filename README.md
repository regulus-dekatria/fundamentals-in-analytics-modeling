# Fundamentals in Analytics Modeling

## About
Course project analyzing the **Hotel Booking Demand Dataset** from [Kaggle](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand). The project covers data cleaning, preprocessing, and predictive modeling for hotel booking cancellations.

## Repository Contents

| File | Description |
|------|-------------|
| `Midterm_Data_Cleaning_and_Preprocessing_with_Pandas.ipynb` | Midterm notebook — data cleaning & preprocessing pipeline |
| `hotel_bookings_cleaned.csv` | Cleaned dataset (87,396 rows, 32 columns) exported from the midterm notebook |

## Dataset
- **Source:** [Hotel Booking Demand](https://www.kaggle.com/datasets/jessemostipak/hotel-booking-demand) by Jesse Mostipak (Kaggle)
- **Original size:** 119,390 rows × 32 columns
- **Cleaned size:** 87,396 rows × 32 columns (after removing duplicates, handling missing values, and standardizing data)
